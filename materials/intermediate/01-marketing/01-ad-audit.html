広告監査190項目チェック

★★ 設定1〜2時間 手動4〜8時間 → 5分 Claude Code Google Sheets 広告CSV
← 一覧に戻る

概要

Google Ads・Meta AdsのデータをエクスポートしたCSVをClaude Codeに渡し、190項目の広告監査チェックリストを自動実行する。手動で4〜8時間かかる作業を5分に短縮。チェック結果はGoogle Sheetsに整形出力され、改善優先度順にソートされる。

必要な準備

Claude Code（インストール済み） Google Ads管理画面へのアクセス Meta Ads Managerへのアクセス Python 3.x（pandas, openpyxl）

設定手順

  • 1
    広告データをCSVでエクスポート

    Google Ads管理画面 → レポート → キャンペーンレポートをCSV形式でダウンロード。Meta Adsも同様に「広告レポート」からCSVエクスポート。

  • 2
    チェックリストファイルを作成

    監査項目を audit-rules.md に定義する。カテゴリ別（品質スコア・入札設定・ターゲティング・クリエイティブ・除外KW等）に190項目を記述。

    # audit-rules.md（抜粋） ## 品質スコア（30項目） - [ ] 全KWの品質スコアが5以上か - [ ] 品質スコア3以下のKWに対して広告文を最適化しているか - [ ] ランディングページとKWの関連性が高いか ## 入札設定（25項目） - [ ] 目標CPA/ROASが設定されているか - [ ] 入札戦略が目標に合致しているか - [ ] 予算の消化率が80〜100%の範囲内か
  • 3
    Claude Codeで監査スクリプトを生成

    Claude Codeを起動し、CSVとチェックリストを渡して監査スクリプトを自動生成させる。

    プロンプト例
    以下のファイルを使って広告監査を自動実行するPythonスクリプトを作成してください。

- google_ads_report.csv：Google Adsのキャンペーンデータ
- meta_ads_report.csv：Meta Adsのキャンペーンデータ
- audit-rules.md：190項目のチェックリスト

各チェック項目を自動判定し、結果を以下の形式でExcelに出力してください：
・合否（○/△/×）
・現在値
・推奨値
・改善優先度（高/中/低）
優先度の高いものから並び替えて出力してください。
  • 4
    スクリプトを実行して結果を確認

    生成されたスクリプトを実行。audit_result.xlsx が出力される。Google Sheetsにアップロードして共有。

  • 5
    定期実行の設定（任意）

    毎週月曜日に自動実行するcronジョブを設定。Slack通知と組み合わせると週次監査が完全自動化される。

    # crontab -e で追加 0 8 * * 1 cd /path/to/project && python audit.py && echo "監査完了"

注意点

  • CSVの列名はプラットフォームによって異なる。スクリプト生成時にCSVのヘッダー行をそのままClaude Codeに貼り付けると精度が上がる。
  • Meta AdsのデータはGDPR等の都合でエクスポートできる項目が制限される場合がある。
  • 190項目すべてをCSVから自動判定できるわけではない。目視確認が必要な項目は「要確認」フラグを立てて区別する。
  • 監査結果ファイルに個人情報や入札金額が含まれる場合は社内共有範囲に注意。
