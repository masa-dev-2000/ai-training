Google/Meta広告 入札・除外KW・クリエイティブ自動化

★★★ 設定半日〜 MCP経由で操作自動化 Google Ads API Meta Ads API MCP
← 一覧に戻る

概要

Google Ads API と Meta Marketing API をMCPサーバー経由でClaude Codeから操作できるようにする。入札額の自動調整・除外キーワードの追加・パフォーマンスの低いクリエイティブの入れ替えを自然言語の指示で実行できる。

必要な準備

Claude Code Google Ads APIアクセス（開発者トークン） Meta Business APIアクセストークン Node.js / Python環境

設定手順

  • 1
    Google Ads API の認証情報を取得

    Google Ads APIセンターで開発者トークンを申請。OAuth2.0でアクセストークンを取得し .env に保存する。

    GOOGLE_ADS_DEVELOPER_TOKEN=your_token GOOGLE_ADS_CLIENT_ID=your_client_id GOOGLE_ADS_CLIENT_SECRET=your_client_secret GOOGLE_ADS_REFRESH_TOKEN=your_refresh_token GOOGLE_ADS_CUSTOMER_ID=1234567890
  • 2
    Meta Ads API のアクセストークンを取得

    Meta for Developersでアプリを作成し、広告アカウントへのアクセス権を持つ長期アクセストークンを発行。

    META_ACCESS_TOKEN=your_long_lived_token META_AD_ACCOUNT_ID=act_1234567890
  • 3
    MCPサーバーを実装する

    Claude CodeからAPI操作を呼び出せるMCPサーバーを作成。主要ツール：get_campaigns、update_bid、add_negative_keyword、pause_creative。

    # mcp-ads-server.py（抜粋） from mcp.server import Server import google.ads.googleads.client as google_ads app = Server("ads-automation") @app.tool() async def update_bid(campaign_id: str, new_bid_micros: int): """キャンペーンの入札額を更新する""" client = google_ads.GoogleAdsClient.load_from_env() # 入札額更新処理... @app.tool() async def add_negative_keyword(campaign_id: str, keyword: str): """除外キーワードを追加する""" # 除外KW追加処理...
  • 4
    Claude CodeのMCP設定ファイルに追加
    # .claude/settings.json { "mcpServers": { "ads-automation": { "command": "python", "args": ["mcp-ads-server.py"], "env": { "GOOGLE_ADS_DEVELOPER_TOKEN": "${GOOGLE_ADS_DEVELOPER_TOKEN}" } } } }
  • 5
    自然言語で広告操作を実行
    指示例
    先週のGoogle Adsキャンペーンを確認して、CPAが目標の150%を超えているキャンペーンの入札額を20%下げてください。また、CTRが0.5%未満のキーワードを除外KWに追加してください。

注意点

  • Google Ads APIの開発者トークン取得には審査が必要（数日〜数週間）。テスト環境では「テストアカウント」を使用すること。
  • 入札額の自動変更は取り消せないため、必ず変更前に確認ステップを挟む設計にする。
  • Meta Ads APIのレート制限に注意。1時間あたりのAPIコール数が制限されている。
  • 本番環境での初回実行は少数のキャンペーンに限定してテストする。
