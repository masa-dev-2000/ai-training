概要

YouTubeのURLを渡すだけで、字幕テキストを取得し、ブログ記事・SNS投稿3本・サムネイル画像のプロンプトを一括生成する。動画制作後のコンテンツ展開作業を大幅に自動化できる。

処理フロー

YouTube URL → 字幕取得 → ブログ記事 + SNS投稿×3 + サムネイル画像

必要な準備

YouTube Data API v3（無料枠で十分） yt-dlp または youtube-transcript-api Claude Code 画像生成AI（Stable Diffusion / DALL-E等）

設定手順

  • 1
    YouTube字幕取得ツールをインストール
    pip install youtube-transcript-api # または pip install yt-dlp
  • 2
    字幕取得スクリプトを作成（Claude Codeで生成）
    プロンプト例
    YouTube URLを引数にとり、字幕テキストを取得して整形するPythonスクリプトを作成してください。
・youtube-transcript-apiを使用
・日本語字幕を優先、なければ英語字幕を取得
・タイムスタンプを除去してテキストのみを返す
・出力：transcript.txt
  • 3
    コンテンツ一括生成プロンプトを実行

    取得した字幕テキストをClaude Codeに渡して4種類のコンテンツを一括生成する。

    コンテンツ生成プロンプト
    添付の字幕テキスト（transcript.txt）をもとに以下の4つを生成してください。

1. ブログ記事（2000〜3000字）
   ・SEOを意識したH2/H3構成
   ・導入文・まとめを含む
   ・読みやすい文体

2. X（旧Twitter）投稿文（140字以内）×3本
   ・それぞれ異なる切り口・フック

3. LinkedIn投稿文（400〜600字）×1本
   ・プロフェッショナルなトーン
   ・動画視聴を促すCTAを含む

4. サムネイル画像生成用プロンプト
   ・16:9比率
   ・動画の核心メッセージを表現
   ・テキストオーバーレイの指示も含む
  • 4
    サムネイル画像を生成する

    出力されたプロンプトをCanva AI・Stable Diffusion・DALL-Eに貼り付けてサムネイル画像を生成する。

プロンプト / 設定例（ケース別）

ブログ記事のみ — 詳細プロンプト
以下の字幕テキストからSEOに強いブログ記事を作成してください。

【字幕テキスト】
（transcript.txtの内容を貼り付け）

【記事の条件】
・文字数：2500〜3500字
・ターゲットキーワード：「〇〇」
・想定読者：〇〇（初心者 / ビジネスパーソン等）
・H2見出し：4〜6個
・各H2のあとにH3を1〜3個追加
・リード文（200字）・まとめ（300字）を必ず含める
・箇条書きや図解の説明を活用して読みやすくする
・動画本編の視聴を促すCTAを本文中に1箇所挿入
SNS3本のみ — プラットフォーム別プロンプト
以下の字幕テキストから、各SNSに最適化した投稿文を作成してください。

【字幕テキスト】
（transcript.txtの内容を貼り付け）

【出力する投稿文】

① X（旧Twitter）投稿×3本（各140字以内）
   - 1本目：数字や実績を使ったフック型
   - 2本目：問いかけ・共感型
   - 3本目：「知らないと損」「意外な事実」型

② Instagram キャプション（300字以内）
   ・ハッシュタグ10個を末尾に追加
   ・改行を多めに使い読みやすく

③ LinkedIn 投稿（500字）
   ・プロフェッショナルなトーン
   ・動画視聴のリンクを誘導する一文を含む
フル一括生成 — 4コンテンツ同時プロンプト
以下の字幕テキストからコンテンツを4種類一括生成してください。

【字幕テキスト】
（transcript.txtの内容を貼り付け）

【生成物①】ブログ記事（2000〜3000字）
・SEO対策済みのH2/H3構成
・リード文・まとめ含む

【生成物②】X（旧Twitter）投稿（140字以内×3本）
・切り口をそれぞれ変えること

【生成物③】LinkedIn投稿（400〜600字）
・プロフェッショナルなトーン

【生成物④】サムネイル画像プロンプト
・16:9比率
・タイトル文字・ビジュアル要素を具体的に指定
・使用する色・フォントスタイルも記述

各生成物は「---」で区切って出力してください。

確認チェックリスト

  • 字幕テキストを目視確認し、明らかな誤字・文脈の切れ目がないか確認したか
  • 自動生成された字幕（CC）の場合、固有名詞・専門用語が正しく取得されているか確認したか
  • ブログ記事に動画の文脈が正しく反映されているか（視聴後に確認することを推奨）
  • SNS投稿の文字数が各プラットフォームの制限内に収まっているか確認したか
  • サムネイル画像プロンプトで生成した画像にテキストの誤字がないか確認したか
  • 対象が自社コンテンツ（または許可を得たコンテンツ）であることを確認したか

注意点

  • 自動生成された字幕（CC）は精度が低いことがある。手動字幕が設定されたチャンネルで使うと精度が上がる。
  • 生成されたブログ記事は必ず加筆修正する。動画の文脈が失われることがあるため、視聴後に編集するのが理想。
  • 他者のYouTube動画を対象にする場合は著作権に注意。自社コンテンツまたは許可を得たコンテンツのみに使用する。
