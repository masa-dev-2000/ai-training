YouTube動画URL→ブログ記事+SNS投稿3本+サムネ画像を一括生成

★★ 設定1〜2時間 動画1本から4コンテンツを自動生成 YouTube Data API Claude Code 画像生成AI
← 一覧に戻る

概要

YouTubeのURLを渡すだけで、字幕テキストを取得し、ブログ記事・SNS投稿3本・サムネイル画像のプロンプトを一括生成する。動画制作後のコンテンツ展開作業を大幅に自動化できる。

処理フロー

YouTube URL → 字幕取得 → ブログ記事 + SNS投稿×3 + サムネイル画像

必要な準備

YouTube Data API v3（無料枠で十分） yt-dlp または youtube-transcript-api Claude Code 画像生成AI（Stable Diffusion / DALL-E等）

設定手順

  • 1
    YouTube字幕取得ツールをインストール
    pip install youtube-transcript-api # または pip install yt-dlp
  • 2
    字幕取得スクリプトを作成（Claude Codeで生成）
    プロンプト例
    YouTube URLを引数にとり、字幕テキストを取得して整形するPythonスクリプトを作成してください。
・youtube-transcript-apiを使用
・日本語字幕を優先、なければ英語字幕を取得
・タイムスタンプを除去してテキストのみを返す
・出力：transcript.txt
  • 3
    コンテンツ一括生成プロンプトを実行

    取得した字幕テキストをClaude Codeに渡して4種類のコンテンツを一括生成する。

    コンテンツ生成プロンプト
    添付の字幕テキスト（transcript.txt）をもとに以下の4つを生成してください。

1. ブログ記事（2000〜3000字）
   ・SEOを意識したH2/H3構成
   ・導入文・まとめを含む
   ・読みやすい文体

2. X（旧Twitter）投稿文（140字以内）×3本
   ・それぞれ異なる切り口・フック

3. LinkedIn投稿文（400〜600字）×1本
   ・プロフェッショナルなトーン
   ・動画視聴を促すCTAを含む

4. サムネイル画像生成用プロンプト
   ・16:9比率
   ・動画の核心メッセージを表現
   ・テキストオーバーレイの指示も含む
  • 4
    サムネイル画像を生成する

    出力されたプロンプトをCanva AI・Stable Diffusion・DALL-Eに貼り付けてサムネイル画像を生成する。

注意点

  • 自動生成された字幕（CC）は精度が低いことがある。手動字幕が設定されたチャンネルで使うと精度が上がる。
  • 生成されたブログ記事は必ず加筆修正する。動画の文脈が失われることがあるため、視聴後に編集するのが理想。
  • 他者のYouTube動画を対象にする場合は著作権に注意。自社コンテンツまたは許可を得たコンテンツのみに使用する。
