画面録画 → キャプション・矢印オーバーレイ動画を自動生成

★★★ 設定半日〜 解説動画の後処理を自動化 FFmpeg Whisper Claude Code
← 一覧に戻る

概要

操作解説の画面録画（音声付き）を入力として、ナレーションから字幕を自動生成・重要な操作箇所に矢印やハイライトボックスのオーバーレイを自動追加してYouTubeやドキュメントに使える解説動画を出力する。

必要な準備

FFmpeg（テキスト・矢印描画のためのdrawtext/drawbox） OpenAI Whisper（字幕生成） Python（Pillow・moviepy） Claude Code

設定手順

  • 1
    Whisperで字幕（SRT）ファイルを生成
    # 字幕ファイルをSRT形式で出力 whisper input.mp4 --language ja --output_format srt --output_dir ./ # input.srt が生成される
  • 2
    字幕を動画に焼き込む（FFmpeg）
    # SRT字幕を動画に焼き込み ffmpeg -i input.mp4 \ -vf "subtitles=input.srt:force_style='FontName=Noto Sans JP,FontSize=20,PrimaryColour=&HFFFFFF&,BackColour=&H80000000&'" \ -c:a copy output_with_subtitles.mp4
  • 3
    強調表示のオーバーレイ情報を生成（Claude Code）

    字幕テキストをClaudeに解析させ、「クリック」「ここを注目」等の操作ポイントのタイムスタンプと位置情報（スクリーンショットから推定）を取得する。

    # overlays.json（Claude Codeで生成） [ {"start": 5.2, "end": 7.8, "type": "arrow", "x": 450, "y": 200, "label": "ここをクリック"}, {"start": 12.0, "end": 15.0, "type": "highlight", "x1": 200, "y1": 150, "x2": 600, "y2": 250} ]
  • 4
    オーバーレイを適用した最終動画を生成
    # Python + moviepy でオーバーレイを合成（Claude Codeで生成） from moviepy.editor import VideoFileClip, CompositeVideoClip from moviepy.video.tools.drawing import color_gradient # overlays.jsonを読み込んでClipとして生成→CompositeVideoClipで合成

注意点

  • オーバーレイの位置（x, y座標）はAIが自動推定するが、精度は動画の解像度と操作の明確さに依存する。重要な動画は座標を手動調整する。
  • FFmpegのdrawtextフィルターは日本語フォントパスを絶対パスで指定する必要がある。
  • moviepyは処理が遅い。長い動画（10分以上）はFFmpegのみで処理する方が速い場合がある。
