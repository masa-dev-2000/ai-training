画面録画 → キャプション・矢印オーバーレイ動画を自動生成

★★★ 設定半日〜 解説動画の後処理を自動化 FFmpeg Whisper Claude Code
← 一覧に戻る

概要

操作解説の画面録画（音声付き）を入力として、ナレーションから字幕を自動生成・重要な操作箇所に矢印やハイライトボックスのオーバーレイを自動追加してYouTubeやドキュメントに使える解説動画を出力する。

最終動画の内容
Whisperが生成した日本語字幕が画面下部に表示され、操作箇所に赤い矢印と「ここをクリック」のラベルが自動表示される。重要な入力欄はオレンジの枠線でハイライト。担当者は後処理なしでYouTubeやSlackに共有できる状態の解説動画が完成する。

必要な準備

FFmpeg（テキスト・矢印描画のためのdrawtext/drawbox） OpenAI Whisper（字幕生成） Python（Pillow・moviepy） Claude Code

設定例（ケース別）

# 字幕のみ：WhisperでSRT生成→FFmpegで焼き込み（最もシンプル） whisper input.mp4 --language ja --output_format srt --output_dir ./ ffmpeg -i input.mp4 \ -vf "subtitles=input.srt:force_style='FontName=Noto Sans JP,FontSize=22,PrimaryColour=&HFFFFFF&,BackColour=&H80000000&,MarginV=30'" \ -c:a copy output_captioned.mp4
# 矢印オーバーレイ：Claude CodeでSRTからoverlays.jsonを生成→Pythonで合成 # 1. SRT生成（字幕のみの場合と同じ） whisper input.mp4 --language ja --output_format srt # 2. Claude Codeに字幕テキストを渡してoverlays.jsonを生成させる # 3. Pythonスクリプトで矢印クリップを合成 python3 apply_overlays.py \ --input input.mp4 \ --overlays overlays.json \ --output output_with_arrows.mp4
# ハイライトボックス：FFmpegのdrawboxフィルターを使う方法 # overlays.jsonの"type":"highlight"エントリをdrawboxフィルターに変換 ffmpeg -i output_captioned.mp4 \ -vf "drawbox=enable='between(t,8.0,11.5)':x=580:y=300:w=340:h=80:color=orange@0.8:t=4" \ -c:a copy output_highlighted.mp4

設定手順

  • 1
    Whisperで字幕（SRT）ファイルを生成
    # 字幕ファイルをSRT形式で出力 whisper input.mp4 --language ja --output_format srt --output_dir ./ # input.srt が生成される
  • 2
    字幕を動画に焼き込む（FFmpeg）
    # SRT字幕を動画に焼き込み ffmpeg -i input.mp4 \ -vf "subtitles=input.srt:force_style='FontName=Noto Sans JP,FontSize=20,PrimaryColour=&HFFFFFF&,BackColour=&H80000000&'" \ -c:a copy output_with_subtitles.mp4
  • 3
    強調表示のオーバーレイ情報を生成（Claude Code）

    字幕テキストをClaudeに解析させ、「クリック」「ここを注目」等の操作ポイントのタイムスタンプと位置情報（スクリーンショットから推定）を取得する。

    # overlays.json（Claude Codeで生成） [ {"start": 5.2, "end": 7.8, "type": "arrow", "x": 450, "y": 200, "label": "ここをクリック"}, {"start": 12.0, "end": 15.0, "type": "highlight", "x1": 200, "y1": 150, "x2": 600, "y2": 250} ]
  • 4
    オーバーレイを適用した最終動画を生成
    # Python + moviepy でオーバーレイを合成（Claude Codeで生成） from moviepy.editor import VideoFileClip, CompositeVideoClip from moviepy.video.tools.drawing import color_gradient # overlays.jsonを読み込んでClipとして生成→CompositeVideoClipで合成

確認チェックリスト

  • FFmpegがインストールされているか（`ffmpeg -version` で確認）
  • Whisperで生成したSRTファイルの日本語字幕が正しく認識されているか
  • 日本語フォントのパスが絶対パスで指定されているか（drawtextフィルター用）
  • overlays.jsonのタイムスタンプ（start/end）が動画の実際の操作箇所に合っているか
  • 矢印・ハイライトの座標（x, y）が画面内に収まっているか（解像度の範囲内か）
  • 字幕と矢印が重なって読みにくくなっていないか（プレビューで確認）
  • 最終動画をYouTube・Slack等の配布先で実際に再生して確認したか

注意点

  • オーバーレイの位置（x, y座標）はAIが自動推定するが、精度は動画の解像度と操作の明確さに依存する。重要な動画は座標を手動調整する。
  • FFmpegのdrawtextフィルターは日本語フォントパスを絶対パスで指定する必要がある。
  • moviepyは処理が遅い。長い動画（10分以上）はFFmpegのみで処理する方が速い場合がある。
