Remotion × Claude Codeでプロンプト1発・1分で動画生成（ロゴアニメ・データビジュアライゼーション・テロップ等）

★★★★ エンジニア向け 1分で動画生成 Remotion React Claude Code

概要

Remotionは「Reactでプログラマブルな動画を作る」ライブラリ。Claude Codeにどんな動画を作りたいか伝えるだけで、Reactコンポーネントとして動画を自動生成し、MP4にレンダリングする。動画制作ソフト不要。

完成イメージ（出力例）

↓ 「売上棒グラフアニメーション動画を作って」と指示した場合の生成コードと動画の説明

Claude Code が生成した src/MyVideo.jsx（抜粋）
import { AbsoluteFill, interpolate, useCurrentFrame, useVideoConfig } from 'remotion';

const data = [100, 150, 120, 200, 180, 250]; // 万円
const labels = ['1月', '2月', '3月', '4月', '5月', '6月'];

export const SalesChart = () => {
  const frame = useCurrentFrame();
  const { durationInFrames } = useVideoConfig();

  return (
    <AbsoluteFill style={{ background: '#fff', padding: 60 }}>
      <h1 style={{ fontFamily: 'Noto Sans JP', fontSize: 32, color: '#0f172a' }}>
        2026年上半期 売上推移
      </h1>
      <div style={{ display: 'flex', alignItems: 'flex-end', gap: 20, marginTop: 40, height: 300 }}>
        {data.map((val, i) => {
          const delay = i * 8;
          const height = interpolate(frame, [delay, delay + 20], [0, val], { extrapolateRight: 'clamp' });
          return (
            <div key={i} style={{ display: 'flex', flexDirection: 'column', alignItems: 'center', gap: 8 }}>
              <div style={{ background: 'linear-gradient(180deg,#3b82f6,#1d4ed8)', width: 60, height: height * 1.1, borderRadius: 4 }} />
              <span style={{ fontSize: 14, color: '#64748b' }}>{labels[i]}</span>
            </div>
          );
        })}
      </div>
    </AbsoluteFill>
  );
};
生成される動画の内容
白背景に「2026年上半期 売上推移」のタイトルが表示され、6本の棒グラフが左から順番に下から上へアニメーションで伸びる。各棒の色はブルー系グラデーション。5秒（150フレーム）でMP4 1280×720に書き出し完了。

Remotionの特長

# Remotionでできること ・ロゴアニメーション（フェードイン・モーション） ・データビジュアライゼーション（数値が動くグラフ・棒グラフ） ・テロップ動画（字幕・タイトルカード） ・プレゼン動画（スライド風のアニメーション） ・ソーシャルメディア用の短尺動画（9:16 / 1:1） # メリット ・コード（JSX）なのでバージョン管理できる ・データをJSONで渡せるため、データが変わっても自動更新 ・フレームごとに精密な制御が可能

設定例（ケース別）

棒グラフアニメーション — Claude Codeへの指示例
Remotionで売上棒グラフアニメーション動画を作成してください。

データ：月別売上 [100, 150, 120, 200, 180, 250]（万円）、1〜6月
サイズ：1280×720、5秒（150フレーム、30fps）
アニメーション：棒グラフが左から順に下から上へ伸びる（各棒に0.3秒の遅延）
デザイン：白背景、棒はブルー系グラデーション、タイトル「2026年上半期 売上推移」（Noto Sans JP）
値ラベル：各棒の上に金額を表示
ロゴアニメーション — Claude Codeへの指示例
Remotionでロゴアニメーション動画を作成してください。

素材：public/logo.svg（会社ロゴ）
サイズ：1920×1080、3秒（90フレーム）
アニメーション：
  0〜30f：ロゴが中央からフェードイン＋スケール 0.8→1.0
  30〜60f：静止（ロゴ表示）
  60〜90f：フェードアウト
背景：ダークネイビー（#0f172a）
キャッチコピー：ロゴ下に「AI で、仕事を変える。」をフェードインで表示
テロップ動画 — Claude Codeへの指示例
Remotionでテロップ動画を作成してください。

テロップ内容（配列で渡す）：
[
  { text: "AIを使えば、作業時間が10分の1に。", start: 0, end: 90 },
  { text: "プロンプト1行で、報告書が完成する。", start: 90, end: 180 },
  { text: "今日から始めるAI活用術", start: 180, end: 270 }
]
サイズ：1080×1920（縦型・リール用）
デザイン：黒背景、白テキスト（Noto Sans JP Bold）、フォントサイズ56px
アニメーション：各テロップが下からスライドインして表示

設定手順

  • 1
    Remotionプロジェクトを作成
    npx create-video@latest cd my-video npm install
  • 2
    Claude Codeで動画コンポーネントを生成
    プロンプト例（数値グラフアニメーション）
    Remotionで以下の動画を生成するReactコンポーネントを作成してください。

動画の内容：
・期間：5秒（150フレーム、30fps）
・サイズ：1280×720（16:9）
・内容：売上データの棒グラフが左から順番にアニメーションで表示される
・データ：月ごとの売上 [100, 150, 120, 200, 180, 250]（万円）
・デザイン：白背景、棒はブルー系グラデーション、タイトル「2026年上半期 売上推移」
・フォント：Noto Sans JP（Google Fontsから読み込み）

必要なファイル：
1. src/MyVideo.jsx（メインコンポーネント）
2. src/Root.jsx（Remotion設定の更新）
  • 3
    プレビューで確認
    npm run dev # ブラウザで http://localhost:3000 を開いてプレビュー
  • 4
    MP4にレンダリング
    # ローカルレンダリング npx remotion render src/index.ts MyVideo output.mp4 # クラウドレンダリング（Lambdaを使う場合） npx remotion lambda render

確認チェックリスト

  • Node.js 18以上がインストールされているか（`node -v` で確認）
  • Remotionプロジェクトが正常に作成できたか（`npm run dev` でブラウザが開くか）
  • 日本語フォント（Noto Sans JP等）がGoogle Fontsから正しく読み込まれているか
  • データをハードコードではなくJSONやpropsで渡す設計になっているか
  • フレームレート・解像度・尺がYouTube・SNS等の用途に合っているか
  • MP4レンダリングが正常に完了したか（出力ファイルを再生して確認）
  • 商用利用の場合はRemotionのライセンス条件を確認したか

注意点

  • Remotionはほぼ何でもできるが、Reactの知識が必要。JSXの基本（コンポーネント・props・CSS-in-JS）が理解できていると理解がスムーズ。
  • 日本語フォントはGoogle Fontsから動的に読み込む方法が安定。ローカルフォントパスを指定する方法もある。
  • ローカルレンダリングはCPU集中的で時間がかかる。1分の動画で数分かかることがある。Lambda等のクラウドレンダリングが速い。
  • 商用利用にはRemotionのライセンス確認が必要（一定以上のユーザー数の場合は有料）。
