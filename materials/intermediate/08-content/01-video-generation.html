Remotion × Claude Codeでプロンプト1発・1分で動画生成（ロゴアニメ・データビジュアライゼーション・テロップ等）

★★★★ エンジニア向け 1分で動画生成 Remotion React Claude Code
← 一覧に戻る

概要

Remotionは「Reactでプログラマブルな動画を作る」ライブラリ。Claude Codeにどんな動画を作りたいか伝えるだけで、Reactコンポーネントとして動画を自動生成し、MP4にレンダリングする。動画制作ソフト不要。

Remotionの特長

# Remotionでできること ・ロゴアニメーション（フェードイン・モーション） ・データビジュアライゼーション（数値が動くグラフ・棒グラフ） ・テロップ動画（字幕・タイトルカード） ・プレゼン動画（スライド風のアニメーション） ・ソーシャルメディア用の短尺動画（9:16 / 1:1） # メリット ・コード（JSX）なのでバージョン管理できる ・データをJSONで渡せるため、データが変わっても自動更新 ・フレームごとに精密な制御が可能

設定手順

  • 1
    Remotionプロジェクトを作成
    npx create-video@latest cd my-video npm install
  • 2
    Claude Codeで動画コンポーネントを生成
    プロンプト例（数値グラフアニメーション）
    Remotionで以下の動画を生成するReactコンポーネントを作成してください。

動画の内容：
・期間：5秒（150フレーム、30fps）
・サイズ：1280×720（16:9）
・内容：売上データの棒グラフが左から順番にアニメーションで表示される
・データ：月ごとの売上 [100, 150, 120, 200, 180, 250]（万円）
・デザイン：白背景、棒はブルー系グラデーション、タイトル「2026年上半期 売上推移」
・フォント：Noto Sans JP（Google Fontsから読み込み）

必要なファイル：
1. src/MyVideo.jsx（メインコンポーネント）
2. src/Root.jsx（Remotion設定の更新）
  • 3
    プレビューで確認
    npm run dev # ブラウザで http://localhost:3000 を開いてプレビュー
  • 4
    MP4にレンダリング
    # ローカルレンダリング npx remotion render src/index.ts MyVideo output.mp4 # クラウドレンダリング（Lambdaを使う場合） npx remotion lambda render

注意点

  • Remotionはほぼ何でもできるが、Reactの知識が必要。JSXの基本（コンポーネント・props・CSS-in-JS）が理解できていると理解がスムーズ。
  • 日本語フォントはGoogle Fontsから動的に読み込む方法が安定。ローカルフォントパスを指定する方法もある。
  • ローカルレンダリングはCPU集中的で時間がかかる。1分の動画で数分かかることがある。Lambda等のクラウドレンダリングが速い。
  • 商用利用にはRemotionのライセンス確認が必要（一定以上のユーザー数の場合は有料）。
