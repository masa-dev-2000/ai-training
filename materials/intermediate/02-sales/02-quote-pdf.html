「見積書作って」でPDF見積書を出力（スキルコマンド化）

★★ 設定1〜2時間 見積書作成を自然言語で Claude Code Skills WeasyPrint / Puppeteer
← 一覧に戻る

概要

「A社にコンサル料50万円の見積書を作って」という自然言語の指示だけで、正式な体裁のPDF見積書を自動生成するスキルコマンドを構築する。消費税計算・見積番号の自動採番・会社印影の埋め込みまで自動化できる。

↓ /quoteコマンド実行後に生成されるPDF見積書の内容例

必要な準備

Claude Code Python（weasyprint または jinja2） 自社ロゴ・印影画像（PNG） 会社情報・振込先情報

設定手順

  • 1
    HTMLベースの見積書テンプレートを作成

    Jinja2テンプレートで見積書HTMLを作成。変数部分（顧客名・品目・金額等）をプレースホルダーにしておく。

    <!-- quote_template.html（抜粋） --> <div class="header"> <img src="logo.png" alt="会社ロゴ"> <h1>御見積書</h1> </div> <div class="client">{{ client_name }} 御中</div> <div class="quote-number">見積番号：{{ quote_number }}</div> <table> {% for item in items %} <tr> <td>{{ item.name }}</td> <td>{{ item.quantity }}</td> <td>{{ item.unit_price | yen }}</td> <td>{{ item.total | yen }}</td> </tr> {% endfor %} </table> <div class="total">消費税（10%）：{{ tax | yen }}</div> <div class="total">合計：{{ grand_total | yen }}</div>
  • 2
    PDF変換スクリプトを作成（Claude Codeで生成）
    pip install weasyprint jinja2
    # generate_quote.py（抜粋） from jinja2 import Template from weasyprint import HTML import json, datetime def generate_quote(client_name, items, output_path): quote_number = f"Q{datetime.date.today().strftime('%Y%m%d')}-{get_next_seq()}" tax = sum(i['total'] for i in items) * 0.1 grand_total = sum(i['total'] for i in items) + tax html = template.render( client_name=client_name, items=items, quote_number=quote_number, tax=tax, grand_total=grand_total, issue_date=datetime.date.today() ) HTML(string=html).write_pdf(output_path)
  • 3
    スキルファイルを作成する
    # .claude/skills/quote.md --- name: quote description: 見積書をPDFで生成する --- ユーザーの指示から見積書に必要な情報（顧客名・品目・単価・数量）を抽出し、 generate_quote.pyを実行してPDFを生成してください。 品目情報が不完全な場合は確認を求めてください。
  • 4
    実行例
    使い方
    /quote A社にシステム導入コンサル料50万円、月次保守料5万円×3ヶ月の見積書を作って

プロンプト / 設定例（ケース別）

単一品目 — シンプルな見積書
/quote

顧客名：株式会社〇〇
品目：業務改善コンサルティング
単価：800,000円
数量：1式
有効期限：30日
備考：（特記事項があれば）

消費税10%を自動計算して見積書PDFを生成してください。
ファイル名：quote_〇〇_20260515.pdf
複数品目＋値引き — 値引き額を明示した見積書
/quote

顧客名：〇〇工業株式会社
品目一覧：
  1. システム設計・要件定義　　500,000円 × 1式
  2. 開発・実装（基本機能）　　1,200,000円 × 1式
  3. テスト・品質検証　　　　　300,000円 × 1式
  4. 導入支援・操作研修　　　　150,000円 × 1式

値引き：上記合計から10%割引（長期取引割引）
有効期限：2026年6月30日

値引き行を小計の前に明示し、
税抜小計→値引き額→値引き後小計→消費税→税込合計の順で表示してください。
月額SaaS型 — 初期費用＋月額料金の見積書
/quote

顧客名：〇〇株式会社
契約形態：月額サブスクリプション（1年契約）
品目一覧：
  1. 初期導入費用（設定・データ移行）　200,000円 × 1式
  2. 月額利用料（プレミアムプラン）　　 50,000円 × 12ヶ月

契約期間：2026年6月1日〜2027年5月31日
支払い条件：初期費用は契約時一括、月額は毎月月末払い

月額と年間合計を分けて表示し、
「1年間の総費用：〇〇円（税込）」を備考欄に記載してください。

送付前チェックリスト

  • 顧客名（会社名・御中の表記）に誤りがないか
  • 見積番号が連番で管理されており、重複がないか
  • 品目名・単価・数量・合計の計算が正しいか（特に消費税率）
  • 値引きがある場合、値引き後の金額が正しく反映されているか
  • 有効期限が設定されており、過去の日付になっていないか
  • 振込先情報（銀行名・支店名・口座番号・名義）に誤りがないか
  • ロゴ・印影が正しく表示されているか（PDF表示で確認）

注意点

  • 見積番号の採番はファイルまたはDBで管理し、重複しないよう連番管理を実装する。
  • WeasyPrintは日本語フォントの設定が必要。システムにNoto Sansや源ノ角ゴシックをインストールしておく。
  • 生成されたPDFは送付前に必ず金額・顧客名・有効期限を目視確認する。
