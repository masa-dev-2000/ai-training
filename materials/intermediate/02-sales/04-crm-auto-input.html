商談メモ → CRM自動入力 ＋ 次アクション提案

★★★ 設定半日〜 商談後の入力作業を自動化 HubSpot API Claude Code
← 一覧に戻る

概要

商談後に書いたメモ（箇条書きでもOK）をClaude Codeに渡すと、構造化してHubSpot / SalesforceのCRMに自動入力し、商談フェーズと会話内容に基づいて次のアクションを提案する。商談後の入力作業（平均20〜30分）を3分に短縮する。

必要な準備

HubSpot アカウント（無料プランでも可） HubSpot Private App APIキー Claude Code Python（hubspot-api-client）

設定手順

  • 1
    HubSpot Private App を作成してAPIキーを取得

    HubSpot → 設定 → インテグレーション → プライベートアプリ → アプリ作成。スコープ：crm.objects.contacts.write、crm.objects.deals.write、crm.objects.notes.write

  • 2
    MCPサーバーまたはスクリプトを作成（Claude Codeで生成）
    プロンプト例
    HubSpot APIを使って商談メモから以下を自動処理するPythonスクリプトを作成してください。

処理内容：
1. 商談メモのテキストから構造化データを抽出
   （会社名・担当者・課題・予算・決裁者・競合・次回アクション・確度）
2. HubSpotの対応するコンタクト・案件を検索or作成
3. アクティビティノートとして記録
4. 案件フェーズを更新
5. 次のアクション候補3つを提案（内容・期限・優先度付き）

HubSpot APIキー：環境変数から取得
  • 3
    スキルコマンド化する
    # .claude/skills/crm-update.md --- name: crm-update description: 商談メモをCRMに自動入力して次アクションを提案する --- ユーザーの商談メモからcrm_updater.pyを実行し、 HubSpotへの自動入力と次アクション提案を行ってください。
  • 4
    実行例
    使い方
    /crm-update
今日の商談メモ：
・株式会社〇〇 田中部長と面談（製造業、300名）
・現在の課題：月次報告書作成に丸2日かかっている
・予算：年間100〜200万で検討中
・決裁は田中部長が単独でできる
・競合は〇〇社と比較中、来週デモを見せてほしいとのこと
・確度：B（先方は前向き）

注意点

  • 顧客の個人情報・商談金額はClaude Codeのプロジェクト設定でデータ保持をオフにしてから使用する。
  • HubSpotの既存データと重複しないよう、コンタクト検索ロジック（メールアドレス・会社名）を丁寧に実装する。
  • Salesforceを使う場合はsimple-salesforceライブラリを使用。APIスコープはHubSpotと異なるため調整が必要。
