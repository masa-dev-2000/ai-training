## 参加者の反応・発言から見える気づき

### 良かった点

1. **「情報漏洩リスク」と「学習設定」の話に最も強く反応**
   - 「意外と情報漏れますよというのが改めてそう思った」
   - 「企業向けプランだとそうなんだ」「学習に使わない設定があるのは勉強になった」
   - テスター1と同様、この部分が最大の学び

2. **ハルシネーションの実例が響いた**
   - 「でもこんなに間違うことあるんすね。全然嘘つくことあるんですね」
   - 設計の専門分野で「平気でこんな間違いする時ある」「専門家からしたらそんなことないだろうって思うこと平気で言い始める」と具体的な経験を共有
   - → **業界別のハルシネーション実例集があると説得力が増す**

3. **AI間の比較に興味**
   - ChatGPTとGeminiの回答の違いに「面白い」と反応
   - 「チャットGPTは嘘つきです」「Geminiの方がまだ嘘つかない」というAIごとの癖の話に強い関心
   - Claudeの紹介（倫理観、嘘つかなさ）にも「じゃあそっち使ってみよう」と即反応
   - → **AI比較の情報は中級者に刺さる**

4. **Perplexityの紹介が効果的**
   - 情報のソース（出典元）を表示するAIとしてPerplexityを紹介
   - → 事実確認の具体的手段として有効

5. **コピーボタンの評価が高い**
   - 「コピーとかもちゃんと作ってくださってるからすごい楽でしたよ」
   - ただし「コピーしてどうやって貼り付けるのっていうところさえできればスムーズ」
   - → **コピー＆ペーストの手順を丁寧に案内する必要あり**

6. **AI＝「優秀な新人」の例えが非常に響いた**
   - 「めっちゃ優秀な新人と思って相手しなきゃだめですよね」
   - 「いつ会社をやめて暴露されるかわかんない」「毎日記憶を忘れする新人」
   - → **この例えは今後も使える。講義資料に正式採用すべき**

### 改善が必要な点

1. **中級者には内容が簡単すぎた**
   - 実質43分で全内容が終了（90分想定の半分以下）
   - 自己紹介文ワークはほぼ飛ばした
   - 「僕の全ては終わったので次からは新しい情報かと思います」と発言
   - → **中級者向けの発展コンテンツが絶対に必要**

2. **第2回との境界問題が再浮上**
   - 「第2回はこの辺を詳しくやる回にしようかなと思ってるけど、内容かぶってちょっとつまらないかな」
   - テスター2のレベルだと第1回と第2回の差が見えにくい
   - → **第2回は明確に「型」の体系化＋実務演習にフォーカスすべき**

3. **人数が増えた場合の時間管理が課題**
   - 「人数が多くなってきた時にわちゃわちゃするかも」
   - 「パソコンでコピーアンドペーストってやるんだみたいなところが始まると大変そう」
   - → **参加者のITリテラシーによって大幅に時間が変わる。バッファが必要**

4. **情報のソース表示（出典元）の機能をもっと強調すべき**
   - 事実確認の具体的手段としてPerplexityを紹介したが、資料には入っていない
   - 「リンク出させてもそのリンクたどってったらないじゃんとかある」という実体験も
   - → **事実確認の具体的手段（出典表示の指示方法、Perplexity等）を資料に追加**

5. **企業版AIの価格情報の質問が出た**
   - 「企業版っていうのはなんかすげえ高いやつですか？」
   - Gemini約4000円/月、Claude 100ドル/月プランの話
   - → **参考として価格帯の情報があると判断材料になる**
