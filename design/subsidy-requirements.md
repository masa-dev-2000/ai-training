# 人材開発支援助成金 ― AI研修での活用ガイド

## 推奨コース

**事業展開等リスキリング支援コース**が最有力。

| 項目 | 内容 |
|------|------|
| 助成率（経費） | 中小企業 75%、大企業 60% |
| 賃金助成 | 中小企業 1,000円/時間、大企業 500円/時間 |
| 上限（1人1訓練） | 10〜100時間未満: 30万円（中小） |
| 年間上限 | 1億円/事業所 |
| 期限 | **令和8年度（2026年度）まで**の期間限定 |

---

## 現カリキュラムとの適合チェック

### 最低訓練時間：10時間以上（OFF-JT）

**現在の設計**: 90分 × 8回 = **12時間** → ✅ クリア

ただし余裕が少ない。以下のリスクあり：
- 受講率8割以上が必要 → 1回でも欠席すると8割を切る可能性
- 90分の中に雑談・休憩が含まれると実訓練時間が減る

**対策案**:
- 1回あたり90分→120分に拡大（合計16時間）
- または回数を10回に増やす（合計15時間）
- 休憩時間を訓練時間に含めない設計を明確化

### OFF-JTの要件

| 要件 | 現在の状況 | 対応 |
|------|-----------|------|
| 通常業務から完全に離れて実施 | ✅ オンラインライブ研修 | 所定労働時間内に実施が必須 |
| 所定労働時間内に実施 | ⚠ 要確認 | 勤務時間中に受講させる必要あり |
| eラーニングの場合はHP掲載 | ✅ GitHub Pagesで公開済み | URL記録を保存 |
| テレワーク実施 | ✅ 就業規則の提出不要（2025年改正） | オンライン研修はOK |

### 対象外となる訓練（注意）

以下に該当すると**助成対象外**：
- ビジネスマナー研修
- 法定講習（安全衛生教育等）
- **趣味・教養レベルの内容**

**→ カリキュラムが「趣味・教養」と見なされないように注意が必要。**

「業務に直接活用するためのAIスキル訓練」であることを明確にする必要がある。

## 不正受給にならないための設計ポイント

### 1. 訓練記録を厳密に残す

| 記録すべき項目 | 方法 |
|---------------|------|
| 出席記録 | 各回の開始・終了時刻、参加者名簿 |
| 訓練内容の実施記録 | 各回のカリキュラム実施内容（計画との対照） |
| 受講証明 | 各回のセルフチェック結果、成果物のスクリーンショット |
| 講師の記録 | 講師名、資格・経験の証明書類 |

**→ presentation.htmlのログ機能（Lキーでエクスポート）が活用可能**

### 2. 計画と実施の一致

不正受給の最大リスクは**計画と実施内容の不一致**。

- 提出したカリキュラムの通りに実施する
- 変更がある場合は必ず**変更届**を提出
- 「第1回でやる予定だったことを第2回に変更」等も変更届が必要

### 3. 経費の全額自己負担を証明

- 訓練機関への支払い → 振込記録を保存
- **キックバック・割引は厳禁**（検査で約3割が不正認定される最大要因）
- 領収書・請求書・振込明細をセットで保管

### 4. 受講率8割以上の確保

**現在の8回制の場合**: 1回欠席で出席率87.5%（ギリギリ）、2回欠席で75%（アウト）

**対策**:
- 補講の仕組みを設計しておく
- 録画を残し、欠席者にはアーカイブ視聴＋課題提出で補完
- 出席管理を厳密に（開始時のチャット投稿等で記録）

## カリキュラムへの反映事項

### 「趣味・教養」と見なされないための工夫

現在のカリキュラムで以下の部分がリスク：

| リスク箇所 | 理由 | 対策 |
|-----------|------|------|
| チャレンジ宿題「献立を考えて」 | 業務と無関係 | **業務に関するお題に変更**する or 「指示文の練習として」の位置づけを明確化 |
| 旅行先のおすすめを聞く演習 | 業務と無関係 | 業務関連のお題に変更する |
| 自己紹介文の作成 | 業務との関連が弱い | 「ビジネスシーンでの自己紹介」と明記 |

**全体的な対策**:
- 各回の学習目標を「業務で活用できるスキル」として記述
- 演習のお題を「業務シーン」に寄せる（メール作成、議事録、報告書、企画書等）
- カリキュラムの冒頭に「本訓練の目的：業務におけるAI活用スキルの習得」を明記

### カリキュラムの形式要件

計画届に添付するカリキュラムには以下が必要：

```
訓練カリキュラム
───────────────────
訓練名: AI活用実践研修 初級編
訓練目標: 業務におけるAI（生成AI）活用スキルの習得
対象者: 全社員（職種不問）
訓練期間: 2026年○月○日 ～ 2026年○月○日
総訓練時間: ○○時間

【科目・時間配分】
第1回（○月○日）○:○○〜○:○○ 計○時間
  科目: AIの基礎理解と業務活用の基本
  内容: AIの仕組み、入力ルール（情報セキュリティ）、基本操作

第2回（○月○日）○:○○〜○:○○ 計○時間
  科目: 効果的な指示の出し方
  内容: 指示文の型（役割・背景・出力形式）、実践演習

...（以下各回同様）
```

### 講師の要件

- **外部講師の場合**: OFF-JT部外講師要件確認書の提出が必要
- **社内講師の場合**: 対象分野の知識・経験を有する者
  - AI関連の業務経験、資格、研修歴等を証明できる書類を準備
  - 講師が受講者と同一人物でないこと

## 申請スケジュール

```
[今すぐ]
  ├─ 職業能力開発推進者の選任（労働局に届出）
  ├─ 事業内職業能力開発計画の策定
  └─ カリキュラムの助成金対応版を作成

[訓練開始の1〜2ヶ月前]
  └─ 計画届を労働局に提出
      ├─ 職業訓練実施計画届（様式第1-1号）
      ├─ 訓練カリキュラム
      ├─ 訓練対象者一覧（様式第3号）
      ├─ 事前確認書（様式第11号）
      ├─ 講師要件確認書（外部講師の場合）
      └─ 契約書・見積書

[訓練実施中]
  └─ 各回の記録を厳密に保存
      ├─ 出席記録（開始・終了時刻）
      ├─ カリキュラム実施内容
      └─ 受講者の成果物

[訓練終了後2ヶ月以内]
  └─ 支給申請書類を提出
      ├─ 支給申請書（様式第4号）
      ├─ 経費・賃金の内訳
      ├─ 実施状況報告書
      ├─ 出席記録・受講証明
      └─ 領収書・振込記録
```

## 次のアクション

1. **社労士への相談**（強く推奨）― 不正受給率が約3割と高く、専門家の支援が重要
2. **カリキュラムの助成金対応版を作成** ― 演習お題を業務寄りに変更
3. **訓練時間の再設計** ― 10時間に余裕を持たせる（120分×8回 or 90分×10回）
4. **出席記録・訓練記録の仕組みを整備** ― テンプレート作成
5. **講師の資格・経験証明書類を準備**
6. **計画届の提出スケジュールを逆算**
