# 第5〜8回 設計書

## 第5回：深く調べてまとめる

| 項目 | 内容 |
|------|------|
| スキル | 調べる力の拡張 |
| ゴール | NotebookLMで複数資料を体系的に調査し、出典付きレポートを作成できる |
| 時間 | 90分（9ページ） |

**深堀りポイント:**
- NotebookLMの「自分の資料だけを根拠にする」特性を活かす
- ディープリサーチでレポート形式の出力を取得
- Perplexityとの使い分け（出典確認用）
- 出典の信頼性を自分で判断する実践
- 業種別調査テーマ: A=認知症見守り先進事例 / B=建設業働き方改革 / C=インバウンド対応

## 第6回：AIと一緒に考える

| 項目 | 内容 |
|------|------|
| スキル | 考える力の拡張 |
| ゴール | AIと壁打ちして企画の骨子を作り、弱点指摘も受けられる |
| 時間 | 90分（9ページ） |

**深堀りポイント:**
- 「アイデアを広げる→絞る→構成する→批判させる」の4段階プロセス
- 10個のアイデアから3個に絞る判断力
- 企画の骨子を「背景→課題→提案→効果→リスク」で構造化
- AIに弱点を指摘させる/反論を想定させる/上司の質問を予測させる
- 業種別テーマ: A=子育て×高齢者イベント / B=安全管理強化 / C=常連客増加施策

## 第7回：人に伝わる資料を作る

| 項目 | 内容 |
|------|------|
| スキル | まとめる力 |
| ゴール | 発表資料の構成をAIに作らせ、見せ方を仕上げられる |
| 時間 | 90分（9ページ） |

**深堀りポイント:**
- 「1ページ1メッセージ」の原則
- アウトライン→各ページの中身→レビューの3段階
- AIに「この資料を見た人にどう伝わるか」をレビューさせる
- 画像生成（Gemini）を使ったビジュアル作成
- 業種別テーマ: A=社協の事業紹介 / B=品質管理体制紹介 / C=新規出店エリア分析

## 第8回：自分の業務で実践する（卒業課題）

| 項目 | 内容 |
|------|------|
| スキル | 5つの力すべて（複合） |
| ゴール | 自分の業務課題を選び、調査→整理→企画→文書化→資料骨子を60分で完遂 |
| 時間 | 90分（8ページ） |

**深堀りポイント:**
- 全8回の5つのスキルを1分ずつ復習
- 卒業課題：60分で一気通貫（調査15分→整理+企画15分→文書+資料15分→発表15分）
- 成果発表＋相互フィードバック
- 5つの力の最終セルフチェック
- 「研修前の自分」と「今の自分」の変化を言語化
- 今後のアクションプラン（具体的に何をいつやるか）
