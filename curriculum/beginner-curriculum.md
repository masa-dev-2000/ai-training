# AI研修 初級編カリキュラム

## 研修概要

| 項目 | 内容 |
|------|------|
| 研修名 | AI活用 初級編 ― 今の自分をアップグレードする |
| 対象者 | 一般ビジネスパーソン（職種不問・AI経験不問） |
| 人数 | 10名以下（少人数制） |
| 形式 | オンラインライブ（Zoom/Meet等） |
| 時間 | 90分 × 全8回 |
| 使用ツール | Google Gemini（無料版）、NotebookLM、Google Workspace連携 |

## コンセプト

> **「人を減らす」のではなく「今いる人のパフォーマンスを最大化する」**

AIを使って単純作業を減らすことが目的ではない。AIを「もう一人の優秀なチームメイト」として活用し、一人ひとりのクリエイティブな成果を増やすことが本研修のゴール。

---

## 卒業時の姿（初級編修了時のゴール）

> **自分の業務の中で「ここはAIと一緒にやろう」と判断し、Geminiを使って"自分ひとりでは出せなかった品質・スピードのアウトプット"を自然に出せる人になっている。**

### 卒業時にできていること（5つの力）

| # | 力 | 具体的な状態 |
|---|-----|-------------|
| 1 | **対話力** | AIに意図を正確に伝え、対話を重ねて望む結果を引き出せる |
| 2 | **文章力の拡張** | メール・議事録・報告書のドラフトをAIで作成し、自分の言葉で仕上げられる |
| 3 | **情報収集・分析力** | 調査ツール（ディープリサーチ・NotebookLM）で体系的に調べ、要点を整理してまとめられる |
| 4 | **企画・発想力の拡張** | AIと壁打ちしてアイデアを広げ、企画や提案の骨子を形にできる |
| 5 | **判断力** | AIの出力をうのみにせず、正しさ・適切さを自分で判断して使いこなせる |

### 8回を終えたらできること（具体的なユースケース）

| 分野 | できること |
|------|-----------|
| 文章作成 | メール・議事録・報告書の下書きをAIに作らせ、自分の言葉で仕上げる |
| 情報収集 | 膨大な資料を読み込ませて要約・比較・分析する |
| 企画・発想 | AIと壁打ちして企画や提案の骨子を形にする |
| 発表・共有 | 発表資料の構成や画像をAIと一緒に作る |
| 日常生活 | 献立・旅行計画・お礼文など暮らしの困りごとにも活用する |
| 業務効率化 | 議事録の自動文字起こし→要約、アンケート設計など定型業務を加速する |

### 卒業チェック（第8回で確認）

受講者が以下の課題を**一人で完遂**できれば初級編修了：

> 「自分の実際の業務テーマを1つ選び、AIを活用して
> **調査 → 情報整理 → 企画構成 → 文章化 → プレゼン資料の骨子作成**
> までを90分で完成させる」

## 全8回の構成 ― 卒業時の姿からの逆算

各回に **セッションゴール**（この90分で到達する具体的な状態）と、それが卒業時の5つの力のどれに対応するかを紐づける。

```
【卒業時の姿】
  自分の業務で AI と協働し、一人では出せなかった成果を出せる
    ↑
  ┌─────────────────────────────────────────────┐
  │ 第8回：全スキルを統合して業務課題を解決する    │ ← 総合
  ├─────────────────────────────────────────────┤
  │ 第7回：見せ方をつくる（プレゼン・画像）       │ ← 企画・発想力
  │ 第6回：アイデアを形にする（企画・ブレスト）   │ ← 企画・発想力
  ├─────────────────────────────────────────────┤
  │ 第5回：深く調べる（リサーチ・NotebookLM）     │ ← 情報収集・分析力
  │ 第4回：情報を整理する（要約・比較・分析）     │ ← 情報収集・分析力
  ├─────────────────────────────────────────────┤
  │ 第3回：文章を味方にする（メール・議事録）     │ ← 文章力の拡張
  │ 第2回：伝え方で結果が変わる（プロンプト基本） │ ← 対話力
  │ 第1回：AIと話してみよう（はじめての対話）     │ ← 対話力 + 判断力
  └─────────────────────────────────────────────┘
```

### 第1回：AIと話してみよう ― はじめての対話

**テーマ**: 心理的ハードルを下げる・AIの「手触り」を知る

| 項目 | 内容 |
|------|------|
| 対応する力 | 対話力 / 判断力 |
| 講義:演習 | 3:7（体験重視） |

**セッションゴール（この90分で到達する状態）**:
- ✅ Geminiに自分の言葉で質問し、返ってきた回答を読んで「使える／使えない」を判断できる
- ✅ 聞き方を変えると回答が変わることを体験している
- ✅ AIは間違えることがあると知り、「確認する癖」の必要性を理解している

**ハンズオン内容**:
1. Geminiを開いて好きなことを聞いてみる（自由体験）
2. 聞き方のコツ（具体的に・背景を伝える・やりとりを重ねる）をメール題材で体験
3. 自己紹介文を条件を変えながら完成させる
4. AIの「知ったかぶり」をクイズで体験（事実確認の重要性）

**振り返りで確認すること**:
- 振り返りクイズ3問（OK/NG、AIの正確性、聞き方の効果）
- セルフチェック（テンプレートをコピーしてチャット投稿）

**発展トピック（中級者向け）**:
- 複数AIの比較（同じ質問をChatGPT/Claude/Geminiに投げて回答を比較）
- AIに「聞き出してもらう」手法（インタビュー形式）＋音声入力の活用
- Geminiの拡張機能（Gmail連携、Googleマップ連携）を試す

### 第2回：伝え方で結果が変わる ― 指示文（プロンプト）の型

**テーマ**: AIへの指示の「型」を学び、再現性のある結果を出す

**第1回との違い**: 第1回では「なんとなく良い聞き方」を体験した。第2回ではなぜ良い結果が出たのかを「型」として体系化し、再現できるようにする。

| 項目 | 内容 |
|------|------|
| 対応する力 | 対話力 |
| 講義:演習 | 3:7 |

**セッションゴール**:
- ✅ 「役割指定」「背景説明」「出力形式の指定」の3要素を使った指示文を書ける
- ✅ 指示文を改善して回答の質が上がる体験をしている
- ✅ 「1回で完璧を求めず、対話で磨く」という使い方を理解している

**ハンズオン内容**:
1. 指示文の「型」（役割＋背景＋形式）を使って業務系の指示を出してみる
2. 同じお題で指示文を段階的に改善するワーク
3. 参加者同士で指示文を交換して結果を比較する

**振り返りで確認すること**:
- 「指示文の3要素（役割・背景・出力形式）を説明できるか？」
- 「最も結果が変わった指示文はどれ？」

**発展トピック（中級者向け）**:
- 例示を入れて精度を上げるテクニック（Few-shot）
- カスタム指示（自分専用の前提条件を設定する）

**テーマ**: 日常的な文章業務をAIで加速する

| 項目 | 内容 |
|------|------|
| 対応する力 | 文章力の拡張 |
| 講義:演習 | 3:7 |

**セッションゴール**:

**ハンズオン内容**:
1. お題のシチュエーションでビジネスメールを生成→推敲
2. サンプル会議メモから議事録のたたき台を作成

**振り返りで確認すること**:

**発展トピック（中級者向け）**:

```
           【卒業時の姿】

                ↑
  ━━━━━━━━━ 統合フェーズ ━━━━━━━━━
  第8回  全スキルを統合して卒業課題を完遂    [5つの力すべて]
  ━━━━━━━ クリエイティブ活用 ━━━━━━━
  第7回  プレゼン資料・画像で「見せ方」を作る [企画・発想力]
  第6回  AIとブレスト・企画構成             [企画・発想力]
  ━━━━━━━ 情報活用フェーズ ━━━━━━━━
  第5回  リサーチ・NotebookLMで深く調べる   [情報収集・分析力]
  第4回  要約・比較・分析で情報を整理       [情報収集・分析力]
  ━━━━━━━ 基礎スキル構築 ━━━━━━━━━
```

**フェーズの設計意図**:

- 第4〜5回: **情報活用** ― インプットの質とスピードを飛躍的に向上
- 第6〜7回: **クリエイティブ活用** ― 企画・発想・ビジュアルなど付加価値領域へ
- 第8回: **統合** ― 全スキルを組み合わせて実務で成果を出す

## 各回共通の進行構造（90分）

| 時間 | パート | 内容 |
|------|--------|------|
| 0:00-0:05 | オープニング | 今日のゴール提示＋前回の振り返り確認 |
| 0:05-0:20 | ミニ講義 | 今日のテーマの要点（最小限の知識インプット） |
| 0:20-0:70 | ハンズオン | 実際にAIを使う演習（2〜3セクション構成） |
| 0:70-0:80 | 共有タイム | 参加者同士で成果や気づきをシェア |
| 0:80-0:90 | 振り返り＆次回予告 | セルフチェック・気づきの言語化・次回への橋渡し |

### 振り返りの仕組み（全回共通）

毎回、3ステップで振り返りを実施する：

1. **セルフチェック**: その回のゴールに対して5段階で自己評価（チャット投稿）
2. **気づきの言語化**: 「今日一番の発見」を一言で共有
3. **次への接続**: 「次回までにやってみたいこと」を1つ宣言（任意）

## 評価方法

### 各回の到達度確認
- **セルフチェック**: セッションゴールに対する自己評価（5段階）
- **成果物共有**: ハンズオンで作った成果物を全体に見せる
- **振り返りコメント**: 気づきを一言で共有

### 全体の到達度確認（第8回）
- **卒業課題の完遂**: 一気通貫の成果物を作れたか
- **5つの力の自己評価**: 各力について研修前後の変化を可視化
- **アクションプラン**: 研修後に業務でAIを使う具体的な計画

## 注意事項・ポリシー

- **ハルシネーション**: AIの回答は必ず自分で確認する習慣を第1回から徹底
- **機密情報**: 社内の機密情報や個人情報はAIに入力しないルールを明確化
- **著作権**: AI生成物の著作権・利用ルールについて基本的な注意を共有
- **ツールの進化**: AIツールは頻繁にアップデートされるため、研修内容も随時見直す
